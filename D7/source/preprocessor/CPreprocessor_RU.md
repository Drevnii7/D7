# Работа с макросами подстановки

`$macro$`- развернуть макрос с типом по умолчанию

`@macro@`- развернуть макрос с типом `string`
Если макрос уже имеет тип `string` то это никак не повлияет на его работу

`#define macro_name macro_value` - объявление нового макроса

# Макросы подстановки

#### Работа с кареткой
| Макрос | Тип (@)  |     Формат    |
| :---   | :---:    |      :---     |
| POS    | `string` | line`:`column |
| LINE   | `int`    | line          |
| COLUMN | `int`    | column        |
